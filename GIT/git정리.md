## 1. git 사용법 정리

### 새로 작업하는 공간 시나리오.
<br>

1. 원하는 파일 받아오기

``` git clone https://github.com/goddl717/TIL.git```

2. 브랜치 목록 확인

``` git branch ```

3. 브랜치 생성

``` git branch <브랜치 명> ```

4. 브랜치 선택 

``` git checkout <branch 명> ```

### 커밋, 푸시 정리

1. commit 된 내용 확인

``` git status ```

2. commit 하고 싶은 파일 추가

``` git add <file name> ```

3. commit 된 내용 확인

``` git status ```
 
4. 커밋하기

``` git commit -m "message" ```

5. remote로 push 

``` git push <저장소 명> <branch 명> ```
